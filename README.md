Credit Card Usage Analytics Dashboard

This project presents an interactive Credit Card Usage Analytics Dashboard developed using Microsoft Power BI. The dashboard analyzes credit card transaction data to uncover insights about spending behavior across different cities, genders, and card types.

The objective of this project is to explore credit card usage patterns and provide a clear visual understanding of how transactions vary across demographic and geographic segments.

Key Features

Analysis of total spending by gender

Identification of top cities with the highest credit card usage

Comparison of different credit card types (Gold, Platinum, Signature, Silver)

Visualization of transaction distribution and spending patterns

Interactive filters for gender, city, card type, and date

Dynamic visuals including bar charts, treemaps, maps, and donut charts

Tools & Technologies

Data Visualization: Microsoft Power BI

Data Processing: Microsoft Excel / Python

Dataset: Credit Card Transaction Dataset

Insights Generated

Identification of the most used credit card types

Gender-wise spending distribution

City-wise credit card spending analysis

Transaction trends across different categories

Project Goal

The goal of this dashboard is to demonstrate how data visualization techniques can transform raw financial transaction data into meaningful business insights that help understand customer behavior and spending trends.
